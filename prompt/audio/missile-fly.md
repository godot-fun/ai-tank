生成一个用于 Top-Down 2D 坦克大战游戏的导弹飞行循环音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道即可。
必须提供可无缝循环的 loop 片段。

【用途】
追踪导弹在空中飞行时的循环音效。
从发射后持续到命中或消失。

【时长】
0.3～0.8 秒可无缝循环片段。

【风格定位】
现代复古军事科幻循环音效。
比坦克引擎更尖锐、更「追」；循环点干净，无爆音或相位跳变。
避免大型喷气机写实轰鸣。

【声音设计】
小型火箭推进「嗖—嗖—」脉冲循环，节奏均匀、轮廓清晰。
中高频为主，带轻微气流感。
音量适中，不宜盖过射击与其他战斗音。
loop 首尾相位无缝衔接，无 click/pop。

【听感目标】
逼近、追踪、威胁持续存在。
营造「有东西在追你」的紧张感。

【混音要求】
必须无缝循环（loop point 干净）。
与 shoot-missile 发射音衔接自然（发射音结束后切入本循环）。

【禁止项】
不要大型喷气机轰鸣。
不要爆炸声混入循环。
不要人声或 loop 接缝处的爆音。
