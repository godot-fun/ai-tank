生成一个用于 Top-Down 2D 坦克大战游戏的射击冷却就绪/装填完成音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
坦克射击冷却（fire_cooldown）结束、可再次开火时的 subtle 提示。
可选功能，宜轻量，避免干扰战斗。

【时长】
0.08～0.2 秒。

【风格定位】
现代复古轻量 UI 反馈音效。
极短、不抢戏，瞬态清晰可辨。
避免与 ui-select 完全相同。

【声音设计】
极短「咔」或「叮」，暗示弹膛复位、系统就绪。
中频，音量低于射击音，可略偏金属机械感。
瞬起瞬落，无尾音拖泥带水。

【听感目标】
就绪、可射击。
老玩家可凭此节奏控射，新玩家也可忽略。

【混音要求】
适合与 fire_interval 约 0.3～2.0 秒的武器配合。
默认可做极轻量，或仅用于教学/无障碍模式。

【禁止项】
不要完整装填动画级长音效。
不要与 ui-select 完全相同。
不要每帧可听见的噪音（仅冷却结束触发一次）。
