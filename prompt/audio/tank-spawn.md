生成一个用于 Top-Down 2D 坦克大战游戏的敌方坦克出现音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
新敌方坦克从地图边缘或出生点生成时出现。
对应经典坦克大战敌人刷新的提示音。

【时长】
0.4～0.8 秒。

【风格定位】
现代复古街机提示音效。
带有「警告 / 新威胁出现」性质，轮廓清晰、瞬态分明。
避免语音报幕或完整音乐片段。

【声音设计】
上升音调电子提示音（类似「嘟——叮」），层次分明。
或短促机械舱门打开 + 引擎点火声。
中频为主，结尾明确收束，便于玩家反应。
0.7 秒内完全结束。

【听感目标】
警觉、压迫感上升。
暗示新敌人入场，战斗即将继续。

【混音要求】
与 UI 提示音频段错开，避免混淆。
不宜过长，避免打断战斗节奏。

【禁止项】
不要完整坦克移动循环。
不要语音报幕（如 "Enemy approaching"）。
不要音乐片段或 Lo-Fi 失真。
