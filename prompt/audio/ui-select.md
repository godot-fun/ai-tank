生成一个用于 Top-Down 2D 坦克大战游戏的 UI 菜单选择/移动光标音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
主菜单、暂停菜单、设置界面中移动选项光标时播放。

【时长】
0.05～0.15 秒。

【风格定位】
现代复古 UI 反馈音效。
极简、干净，瞬态清晰；高保真 electronic blip。
避免 whoosh 扫频或完整旋律。

【声音设计】
极短单音 electronic blip「哔」，瞬起瞬落。
中频，柔和不刺耳，无尾音拖泥带水。
体积极小，适合连续快速触发仍保持清晰可辨。

【听感目标】
中性、操作确认、界面导航。
不抢戏，可高频重复。

【混音要求】
与 ui-confirm 明显区分（本音效更轻、更短）。
适合连续快速触发。

【禁止项】
不要完整旋律。
不要 whoosh 扫频。
不要人声或 Lo-Fi bit-crush。
