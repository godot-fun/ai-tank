生成一个用于 Top-Down 2D 坦克大战游戏的关卡通关/胜利音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
清空所有敌人或完成关卡目标时播放。

【时长】
0.8～2.0 秒。

【风格定位】
现代复古街机胜利 fanfare 音效。
明亮、上扬、简短庆祝；高保真电子音色，每个音清晰可辨。
非完整 BGM，非现代 orchestral 管弦乐。

【声音设计】
上升音阶或短促号角式电子旋律（3～5 个音），节奏紧凑。
音色明亮，中高频为主，1～2 秒内收束。
可带极轻「叮铃」点缀，层次清晰。

【听感目标】
成就、 relief、进入下一关的期待。
与 game-over 形成明确正负对比。

【混音要求】
一次性播放。
音量适中，不掩盖后续 UI 操作。

【禁止项】
不要完整长音乐（留给 bgm/）。
不要人声欢呼。
不要过于现代 orchestral 管弦乐。
不要 Lo-Fi 失真。
