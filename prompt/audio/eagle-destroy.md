生成一个用于 Top-Down 2D 坦克大战游戏的基地（老鹰）被摧毁音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
玩家基地 Eagle 被敌方炮弹击中摧毁时播放。
对应游戏失败前的关键事件。

【时长】
0.8～1.5 秒。

【风格定位】
现代复古像素游戏关键事件音效。
保留《坦克大战》基地被毁的标志性悲剧感，高保真、层次清晰。
比 tank-death 更长、更重，但避免核爆级夸张或 muddy 低频。

【声音设计】
较大规模爆炸 + 建筑/基地结构崩塌感，攻击瞬态有力。
可带短促警报残响或下降音调电子音，层次分明。
中低频爆炸为主，尾音略长于普通坦克死亡。
结尾可留 0.2 秒「空」或低沉余韵，衔接 game-over。

【听感目标】
失败、失守、剧情转折点。
玩家应立刻感知「基地没了，局势危急」。

【混音要求】
全曲音量可略大，但仍需防止 clipping。
与 tank-death、explosion 层级分明：更长、更重、更有「基地」结构感。

【禁止项】
不要人声惨叫或语音 "Game Over"。
不要过长哀嚎式音乐。
不要核爆级夸张或 Lo-Fi 失真。
