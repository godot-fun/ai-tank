生成一个用于 Top-Down 2D 坦克大战游戏的 UI 菜单确认/开始音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
菜单中确认选项、开始游戏、继续游戏时播放。

【时长】
0.15～0.35 秒。

【风格定位】
现代复古 UI 确认音效。
比 ui-select 更完整、更有「确定」感；高保真、轮廓清晰。
避免完整 fanfare（留给 level-clear）。

【声音设计】
双音或三音上升确认「叮叮」，每个音清晰可辨。
或短促「咚—叮」组合，中频为主，略比 select 音更饱满。
结尾明确收束，0.3 秒内完全结束。

【听感目标】
确定、启动、进入游戏。
给玩家「按下了重要按钮」的反馈。

【混音要求】
与 ui-select 成对使用。
不宜过长，避免阻塞场景切换。

【禁止项】
不要完整 fanfare（留给 level-clear）。
不要爆炸或枪声。
不要人声或 Lo-Fi 失真。
