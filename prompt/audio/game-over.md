生成一个用于 Top-Down 2D 坦克大战游戏的游戏失败音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
基地被毁后游戏暂停/失败界面出现时播放。
衔接 eagle-destroy 之后。

【时长】
1.0～2.5 秒。

【风格定位】
现代复古街机失败提示音效。
经典《坦克大战》Game Over 式下降音阶 DNA，高保真、轮廓清晰。
不宜过于悲伤或冗长，保持街机节奏与现代音质。

【声音设计】
短促下降电子音阶（如 do-si-la-sol 或类似），每个音清晰可辨。
或「嗡——」低频下坠 + 短促终止音。
整体偏消极、收束，非爆炸；中低频为主，2.5 秒内完全结束。

【听感目标】
挫败、结束、等待重来。
与 level-clear 形成明确正负对比。

【混音要求】
与 BGM 可同时存在但频段错开。
一次性播放，不循环。

【禁止项】
不要完整背景音乐。
不要人声 "Game Over"（除非单独做 voice 资源）。
不要过长（超过 3 秒影响重开节奏）。
不要 Lo-Fi bit-crush。
