生成一个用于 Top-Down 2D 坦克大战游戏的普通炮弹射击音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音，便于引擎裁剪与拼接。

【用途】
玩家或敌方坦克发射标准炮弹时的短促开火声。
对应游戏内基础子弹（BasicBullet）。

【时长】
0.15～0.35 秒。
单次触发，无循环。

【风格定位】
现代复古游戏音效（Modern Retro Game SFX）。
保留《坦克大战 / Battle City》小型坦克炮的识别 DNA，但以高保真、清晰瞬态呈现。
音色干净、层次分明，避免 Lo-Fi 失真、厚重 muffled 或 8-bit 低比特量化感。
不追求写实战争录音，但要有现代独立像素游戏的音质精度。

【声音设计】
短促、干脆的「砰」或「噗」声，攻击瞬态（Attack）立即可辨。
轻微金属炮管共鸣，中低频主体 + 少量高频金属细节点缀。
尾音快速衰减，0.2 秒内完全收束，不拖泥带水。
音量冲击适中，适合高频重复播放仍保持清晰可辨。

【听感目标】
紧张、利落、动作感强。
听起来像小型坦克炮，而非大口径火炮或机关枪。

【混音要求】
峰值适中，预留 headroom，避免 clipping。
与其他射击音效频段可区分，便于混音叠加。
适合 44.1kHz / 16-bit 游戏 SFX 管线。

【禁止项】
不要人声、语音、音乐旋律。
不要长混响或大空间回声。
不要真实战场环境音（飞机、人群、风声）。
不要连续连发机关枪感。
不要 Lo-Fi 滤波、bit-crush 或刻意做旧的磁带噪。
