生成一个用于 Top-Down 2D 坦克大战游戏的道具/强化拾取音效。

【输出规格】
WAV / OGG 游戏 SFX。
采样率 44.1kHz 或 48kHz，16-bit 或以上。
单声道或立体声均可。
前后各留 5～10ms 静音。

【用途】
玩家获得升级道具、加血、武器强化等奖励时播放。

【时长】
0.3～0.6 秒。

【风格定位】
现代复古奖励反馈音效。
明亮、正向、即时反馈；高保真电子音色，每个音清晰可辨。
避免过于复杂的现代 UI 音效（whoosh + 多图层）或完整 BGM 片段。

【声音设计】
短促上升音阶「叮—叮—叮」，层次分明。
或经典「拾取星星」式 electronic blip，中高频明亮，能量感轻量。
尾音快速结束，0.5 秒内完全收束，不拖长。

【听感目标】
收获、变强、鼓励继续战斗。
与受击/失败音效形成正负对比。

【混音要求】
适合偶尔触发，峰值适中。
可与不同道具共用，通过 pitch 区分等级。

【禁止项】
不要完整 BGM 片段。
不要人声 "Power Up"。
不要过于复杂的现代 UI 音效（whoosh + 多图层）。
不要 Lo-Fi bit-crush。
